# Usage
<!-- usage -->
```sh-session
$ npm install -g redhead
$ redhead COMMAND
running command...
$ redhead (-v|--version|version)
redhead/0.0.0 darwin-x64 node-v10.15.0
$ redhead --help [COMMAND]
USAGE
  $ redhead COMMAND
...
```
